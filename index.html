SQL Injection

請將這裡的影片連結替換為您的 YouTube 影片。

嘿，國中生們！你們知道嗎？你們上網輸入的資料，都存放在一個叫做資料庫的地方。而駭客有一種叫做「SQL Injection」的技術，就像一把萬能鑰匙，能打開這些資料庫的大門，偷走你們的資料！

這份教材包就是要帶你們，用最簡單、最有趣的方式，了解：
* 什麼是 SQL？ 和資料庫溝通的語言。
* SQL Injection 怎麼運作？ 駭客如何利用漏洞。
* 如何防範？ 保護自己不被入侵！


學會這些，你們就能成為「資安小偵探」，保護自己的網路安全！準備好了嗎？一起來解鎖 SQL Injection 的秘密吧！
