SQL Injection 的幾種攻擊類型

前面我們知道，SQL Injection 就像是駭客利用漏洞，把惡意的 SQL 語句偷偷塞進你的輸入，讓資料庫誤以為是正常的指令並執行。但你知道嗎？駭客們玩這個遊戲，還有好幾種不同的「招式」喔！

我們來看看幾種常見的 SQL Injection 攻擊種類：

In-band SQL Injection

最簡單並最常見的攻擊種類，把惡意指令直接插入網頁輸入欄位。

這什麼叫「In-band」呢？就是駭客透過原有輸入欄位送出攻擊指令，而資料庫回應也直接在同一個網頁上顯示出來。

想像一下，你假借跟老師借筆的契機刻意問老師一個考題，而老師不假思索的直接在黑板上寫出答案。

Blind SQL Injection

當網頁沒有從資料庫直接顯示結果時，攻擊者將根據答回或時間差異判斷攻擊是否成功。

Out-of-band SQL Injection

該類攻擊需要網站與其他系統有互聯（像 DNS、Email 等）才能成功執行
